# aman-bot (Friendly Discord Bot)

#### Prefix = `aman`

#### Table of Commands
* `help` to get info on how to use bots and contact the devs
* `devs` to get developer info
* `gabe` to say hi to gabe
* `avatar <username>` to pull up user pfp
* `quotes` to return a random quote

#### Example Use:
`aman commands`
